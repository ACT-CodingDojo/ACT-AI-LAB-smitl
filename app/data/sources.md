# Dataset sources

Primary source for Italian wine denominations:
- MASAF: https://www.masaf.gov.it/flex/cm/pages/ServeBLOB.php/L/IT/IDPagina/4625

Verification source:
- eAmbrosia register: https://ec.europa.eu/agriculture/eambrosia/geographical-indications-register/
- eAmbrosia API: https://webgate.ec.europa.eu/eambrosia-api/

This MVP dataset is a curated starter list to constrain LLM output to Italian wines only.
